Get NativeLink up and running in just a few minutes using Docker.

## Prerequisites

<Steps>
  <Step title="Install Docker">
    You need Docker installed on your system:
    - **Linux/macOS**: [Docker Desktop](https://www.docker.com/products/docker-desktop)
    - **Windows**: [Docker Desktop for Windows](https://www.docker.com/products/docker-desktop)
  </Step>
</Steps>

## Start NativeLink

<Steps>
  <Step title="Download the configuration">
    <CodeGroup>
    ```bash Linux/macOS
    curl -O \
        https://raw.githubusercontent.com/TraceMachina/nativelink/v0.7.5/nativelink-config/examples/basic_cas.json5
    ```
    
    ```powershell Windows
    Invoke-WebRequest `
        -Uri "https://raw.githubusercontent.com/TraceMachina/nativelink/v0.7.5/nativelink-config/examples/basic_cas.json5" `
        -OutFile "basic_cas.json5"
    ```
    </CodeGroup>
  </Step>

  <Step title="Run the Docker container">
    <CodeGroup>
    ```bash Linux/macOS
    docker run \
        -v $(pwd)/basic_cas.json5:/config \
        -p 50051:50051 \
        ghcr.io/tracemachina/nativelink:v0.7.5 \
        config
    ```
    
    ```powershell Windows
    docker run `
        -v ${PWD}/basic_cas.json5:/config `
        -p 50051:50051 `
        ghcr.io/tracemachina/nativelink:v0.7.5 `
        config
    ```
    </CodeGroup>
    
    <Note>NativeLink will start and listen on port 50051</Note>
  </Step>

  <Step title="Verify it's running">
    Check the logs for "Ready, listening on":
    
    ```
    Ready, listening on 0.0.0.0:50051
    ```
  </Step>
</Steps>

## What's running?

The `basic_cas.json5` configuration provides:

<CardGroup cols={2}>
  <Card title="Content Addressable Storage (CAS)" icon="database">
    Stores build artifacts using content-based addressing
  </Card>
  <Card title="Action Cache (AC)" icon="bolt">
    Caches action results for faster incremental builds
  </Card>
  <Card title="Remote Execution" icon="server">
    Executes build actions on local workers
  </Card>
  <Card title="Scheduler" icon="calendar">
    Matches actions to available workers
  </Card>
</CardGroup>

All data is stored locally in `/tmp/nativelink/` by default.

## Troubleshooting

<AccordionGroup>
  <Accordion title="Port 50051 is already in use">
    Change the port mapping in the docker run command:
    
    ```bash
    docker run -v $(pwd)/basic_cas.json5:/config -p 50052:50051 ...
    ```
    
    Then configure your build tool to use port 50052.
  </Accordion>

  <Accordion title="Permission denied error">
    On Linux, you may need to run Docker with sudo or add your user to the docker group:
    
    ```bash
    sudo usermod -aG docker $USER
    newgrp docker
    ```
  </Accordion>

  <Accordion title="Container fails to start">
    Check the Docker logs:
    
    ```bash
    docker logs <container-id>
    ```
    
    Common issues:
    - Invalid JSON5 in config file
    - Missing volume mount
    - Port conflict
  </Accordion>
</AccordionGroup>

For more detailed troubleshooting, see the [Troubleshooting guide](/operations/troubleshooting).
